# copick-torch

Torch utilities for [copick](https://github.com/copick/copick)

## Quick demo

`uv run examples/simple_training.py`

## Development

### Install development dependencies

```bash
pip install ".[test]"
```

### Run tests

```bash
pytest
```

### View coverage report

```bash
pytest --cov=copick_torch
```
